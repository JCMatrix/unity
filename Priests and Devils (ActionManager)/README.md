# 总结游戏对象的使用

1. 游戏对象可以通过Unity的Find等API查找，但不推荐这么做；
2. 游戏对象可以通过脚本产生，也可以在图形界面中提前创建；
3. 动作通过游戏对象来执行；
4. 游戏对象通常通过tranform.gameObject或this.gameObject来获取；
